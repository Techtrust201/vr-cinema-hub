# Onboarding casque Meta Quest

1. Préparer le Quest (chargé, Wi‑Fi OK).
2. Activer le mode développeur (Meta Developer Hub / compte org).
3. Connecter le Quest en USB.
4. Accepter la clé RSA sur le casque.
5. Installer l’APK (`adb install -r …`).
6. Autoriser les sources inconnues si demandé.
7. Lancer **VR Cinema Quest**.
8. Lire le code à 6 chiffres affiché.
9. Ouvrir le dashboard web.
10. Casques → Appairer.
11. Saisir le code + un nom métier clair.
12. Ajouter le casque à un groupe (ou assignment direct).
13. Assigner la playlist.
14. Vérifier « Application active » (contact &lt; 2 min).
15. Vérifier `desired` / `applied`.
16. Tester une vidéo.
17. Tester un retrait (vidéo ou groupe) → disparition du menu.
18. Diagnostiquer un casque hors ligne : app ouverte ? Internet ? token valide ?

## Important

**Casque allumé ≠ Application VR active.**
Le dashboard affiche le contact serveur de l’app, jamais l’état d’alimentation.
